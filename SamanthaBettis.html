Describe HTML

HTML language basically sets up a websites functions, and the links or pictures or words on a websites pages. HTML provides how a website is set up with how things are placed on a websites surface, and the foundations of coding to even make it an actual website and not just random words in a box.In coding HTML is the literal beginning and end of a websitew with the beginning being setting it as an actual HTML, and the end having a footer and a slash to tie everything together. This language is universal as well, anyone can use it to make any website they would like from the ground up.

Describe CSS

CSS language is what makes a website pretty, and cohessive for the look you want a website to have. The color shceme is set up in the code, as well as different collums and fonts that can be used through a website. This is what makes a website go from a black white page to an intersesing set up of a page with colors and resized shapes around words set up in HTML's basic codes.Pretty much CSS turns a page from nothing to something, using its own coding language to work above the foundations of HTML.
